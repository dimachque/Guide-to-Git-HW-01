# Оставляем коммит

>### Переходим в контроль версий по колонке слева, либо сочетанием клавиш `Ctrl + Shift + G` (_Windows/Linux_) and `Cmd + Shift + G` (_macOS_). Пишем название коммита, затем отправляем
