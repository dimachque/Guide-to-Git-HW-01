# Добавляем папку в Git

>### Слева в колонке выбираем проводник `Explorer` или переходим на него сочетанием клавиш `Ctrl + Shift + E` (Windows/Linux) или `Cmd + Shift + E` (macOS). Нажимаем `Open Folder` и выбираем папку (не файл).
