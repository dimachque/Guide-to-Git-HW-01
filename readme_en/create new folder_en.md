# Open Folder in IDE

>### On the left of the column, select the Explorer tab or use a combination of keys `Ctrl + Shift + E` _(Windows/Linux)_ or `Cmd + Shift + E` _(macOS)_. Click on `Open Folder` and select a folder _(not a file)_.
